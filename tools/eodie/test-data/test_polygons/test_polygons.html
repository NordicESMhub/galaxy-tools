This composite dataset is composed of the following files:

  • shapefile.shp (Geometry File (shp))
  • shapefile.shx (Geometry index File (shx))
  • shapefile.dbf (Columnar attributes for each shape (dbf))
  • shapefile.prj (Projection description (prj)) (optional)
  • shapefile.sbn (Spatial index of the features (sbn)) (optional)
  • shapefile.sbx (Spatial index of the features (sbx)) (optional)
  • shapefile.fbn (Read only spatial index of the features (fbn)) (optional)
  • shapefile.fbx (Read only spatial index of the features (fbx)) (optional)
  • shapefile.ain (Attribute index of the active fields in a table (ain)) (optional)
  • shapefile.aih (Attribute index of the active fields in a table (aih)) (optional)
  • shapefile.atx (Attribute index for the dbf file (atx)) (optional)
  • shapefile.ixs (Geocoding index (ixs)) (optional)
  • shapefile.mxs (Geocoding index in ODB format (mxs)) (optional)
  • shapefile.shp.xml (Geospatial metadata in XML format (xml)) (optional)
